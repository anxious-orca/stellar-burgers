# Проектная работа 11 и 12

[Макет](<https://www.figma.com/file/vIywAvqfkOIRWGOkfOnReY/React-Fullstack_-Проектные-задачи-(3-месяца)_external_link?type=design&node-id=0-1&mode=design>)

Для запуска приложения выполните:

```
npm run start
```

Для запуска Storybook выполните:

```
npm run storybook
```

Для запуска линтера выполните:

```
npm run lint
```

Для запуска форматтера выполните:

```
npm run format
```

Для запуска cypress тестов выполните:

```
npm run cypress
```

Для запуска jest тестов выполните:

```
npm run test
```

## Важно:

Для корректной работы запросов к серверу необходимо добавить переменную BURGER_API_URL в окружение, файл .env
